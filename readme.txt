Plugin for sending push notifications to iPhone with iCornr.

== Description ==

With this plugin you may send push notifications when you publish a new post to your followers at iCornr that
has enabled push notifications on their iPhones. In order to use this plugin you must create an iCornr account and
get VIP status for it. More information about this can be found at http://www.icornr.com

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the unzipped 'icornrpush/' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go the the settings menu and navigate to the "iCornr Push" page.
4. Set the correct values on the settings page (all info can be found at iCornr.com on your 'My iCornr' page).
